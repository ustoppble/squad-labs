Produto

Tudo que você precisa pra orquestrar agentes — em 10 áreas integradas

Cada área resolve um gargalo real do desenvolvimento com agentes. Do pane isolado ao orquestrador canônico, do arquivo à voz.

01

Panes

Roda N agentes em paralelo — cada um isolado, visível, sob controle

Pare de trabalhar em série. Com Panes, você abre quantos agentes quiser — cada um com modelo, provedor e contexto próprios. Um worker escreve testes enquanto outro faz code review e outro lê documentação. Layout livre: arraste, redimensione, empilhe.

4 terminais. 1 cockpit. Tudo em paralelo.

Inclui

  • ▸Múltiplos panes em paralelo
  • ▸Modelo e provedor por pane
  • ▸Contexto isolado entre panes
  • ▸Layout drag-and-resize
  • ▸Live token e cost meter
  • ▸Browser pane nativo
  • ▸Workspace tabs
02

Squad

Orquestrador que quebra missões e delega automaticamente

O coração do ADE. Você define o objetivo; o Squad cuida do resto. O orquestrador quebra a missão em tarefas com dependências, o delegation gate distribui para os workers certos. Workers reportam via handoff quando concluem — ou escalona para você quando bloqueados.

Você faz a missão. O Squad distribui o trabalho.

Inclui

  • ▸Orquestrador canônico
  • ▸Delegation gate configurável
  • ▸Workers especializados
  • ▸Handoff event-driven
  • ▸Escalation automático
  • ▸Squad por projeto (Catalog)
  • ▸Log de decisões em tempo real
03

Catalog

Marketplace de squads prontos — instale em 1 clique

Não repita configuração. O Catalog guarda squads, skills e configs prontos para instalar em 1 clique. Compartilhe receitas entre projetos e times. Do squad de code-review ao pipeline de deploy — padrões que funcionam, disponíveis na hora.

Configurou uma vez. Reutiliza em todo projeto.

Inclui

  • ▸Marketplace de squads prontos
  • ▸Skills instaláveis
  • ▸Configs por projeto
  • ▸Compartilhamento entre equipes
  • ▸1 clique para ativar
  • ▸Versionamento de receitas
04

Providers

6 providers, 20+ modelos — troque por pane sem reconfigurar nada

Anthropic (Claude), OpenAI (GPT/Codex), Google (Gemini), Groq, Ollama e OpenRouter num único painel. Cada pane usa o modelo certo para sua tarefa — Haiku para leitura rápida, Opus para raciocínio pesado, Ollama para tarefas offline. Tudo junto, sem trocar de ferramenta.

Modelo certo pra cada tarefa. Custo estimado antes de disparar.

Inclui

  • ▸Anthropic Claude (todas as versões)
  • ▸OpenAI GPT + Codex
  • ▸Google Gemini
  • ▸Groq (inferência rápida)
  • ▸Ollama (modelos locais)
  • ▸OpenRouter (aggregator)
  • ▸Custo estimado por modelo
05

MCP Tools

35 tools in-process — sem latência, sem configuração extra

Seus agentes leem arquivos, rodam shell, fazem commit, buscam na web, geram imagem e vídeo — tudo nativo. O servidor MCP roda in-process dentro do Overclock: sem latência de rede, sem configuração extra. Filesystem, shell, browser, git, geração de mídia, pesquisa, memória e APIs externas disponíveis para qualquer agente.

35 tools. In-process. Zero latência de rede.

Inclui

  • ▸Filesystem (read, write, glob, grep)
  • ▸Shell / Bash executor
  • ▸Browser nativo (Playwright)
  • ▸Git (status, diff, commit, push)
  • ▸Geração de imagem (DALL-E, Flux)
  • ▸Geração de vídeo (Kling)
  • ▸Geração de áudio (ElevenLabs)
  • ▸Pesquisa web (Perplexity, Brave)
  • ▸Memória persistente (claude-mem)
  • ▸Supabase, APIs externas
  • ▸35 tools no total
06

Mission Room

Centro de comando: crie, rastreie e retome missões entre sessões

O Mission Room persiste o estado completo da missão: status por etapa (pending/running/done/blocked), log em tempo real por worker, custo acumulado. Feche o Overclock, reabra e continue exatamente onde parou. Crie sub-missões, configure aprovação humana, faça handoff manual quando quiser.

Feche o ADE. Reabra. A missão continua do mesmo ponto.

Inclui

  • ▸Criar e nomear missões
  • ▸Status por etapa (pending/running/done/blocked)
  • ▸Log em tempo real por worker
  • ▸Handoff manual e automático
  • ▸Persistência entre sessões
  • ▸Missão aninhada (sub-missão)
  • ▸Aprovação humana configurável
07

Files

Workspace, editor e session picker integrados

File tree integrado, workspace tabs por projeto, session picker para retomar o contexto exato de qualquer sessão anterior. Drag-and-drop para levar arquivos ao pane certo. Sem perder o fio da meada em sessões longas.

Contexto certo, projeto certo, sessão certa — sem procurar.

Inclui

  • ▸File tree integrado
  • ▸Workspace tabs por projeto
  • ▸Session picker (retomar sessão)
  • ▸Drag-and-drop para panes
  • ▸Contexto de arquivo por pane
  • ▸Busca rápida em arquivos
08

ClockVoice

Pro

Fale com seus agentes sem digitar — push-to-talk nativo

Segure o atalho, fale, deixa o agente trabalhar. Push-to-talk com STT local (Whisper, offline), Groq ou qualquer provider OpenAI-compat. Transcrição em tempo real, prompt vai direto ao pane ativo. ClockVoice é exclusivo do plano Pro.

Voz nativa no ADE. Exclusivo Pro.

Inclui

  • ▸Push-to-talk (atalho configurável)
  • ▸STT local Whisper (offline)
  • ▸Groq STT (cloud rápido)
  • ▸OpenAI-compat STT
  • ▸Transcrição em tempo real
  • ▸Prompt por voz para qualquer pane
09

Settings

Configure o ADE do seu jeito — custo, atalhos, tema, tudo

Limite de custo por sessão, limite de tokens por pane, auto-save de sessão, tema cockpit/dark/light, atalhos de teclado customizáveis, notificações de missão, debug mode por pane. O ADE se adapta ao seu fluxo — não o contrário.

Limite de gasto configurável. Nunca estoura o orçamento sem saber.

Inclui

  • ▸Tema cockpit / dark / light
  • ▸Atalhos de teclado customizáveis
  • ▸Custo máximo por sessão
  • ▸Limite de tokens por pane
  • ▸Auto-save de sessão
  • ▸Notificações de missão
  • ▸Debug mode por pane
10

Auth

Local-first. Licença por máquina. Seu código nunca sai.

Autenticação segura com licença por máquina. Nenhum dado de código vai para servidores Overclock — garantia de privacidade, não promessa. Gerenciamento de plano integrado com upgrade Boost → Pro em 1 clique.

Zero dados de código nos servidores Overclock.

Inclui

  • ▸Licença por máquina
  • ▸Local-first (código não sai)
  • ▸Gerenciamento de plano
  • ▸Upgrade Boost → Pro em 1 clique
  • ▸Multi-dispositivo configurável

Pronto para usar o ADE?

Overclock Boost tem todas as 10 áreas (exceto ClockVoice). Pro adiciona voz nativa.
